# Homebrew Tap for mcp-ticketer

Official Homebrew tap for [mcp-ticketer](https://github.com/bobmatnyc/mcp-ticketer) - Universal ticket management interface for AI agents with MCP support.

## Installation

```bash
brew tap bobmatnyc/mcp-ticketer
brew install mcp-ticketer
```

## Upgrade

```bash
brew update
brew upgrade mcp-ticketer
```

## Uninstall

```bash
brew uninstall mcp-ticketer
brew untap bobmatnyc/mcp-ticketer
```

## Documentation

For documentation and usage, see the [main repository](https://github.com/bobmatnyc/mcp-ticketer).

## Formula Updates

This tap contains the official Homebrew formula for mcp-ticketer. Formula updates are synchronized with PyPI releases.
